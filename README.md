# E-Commerce-Sales-Analytic-System

🔹 Project Highlights

• Designed a relational database with 6 interconnected tables – Customers, Products, Categories, Orders, Order_Items, and Payments.
• Implemented Primary Keys and Foreign Key relationships to maintain data integrity.
• Built analytical queries using JOIN, GROUP BY, and aggregate functions to extract meaningful insights.
• Performed customer behavior and sales trend analysis.
• Generated insights such as top-selling products, monthly revenue trends, and category-wise sales performance.
